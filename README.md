# HardCode

### Requirements
1. node and npm
2. docker
3. MongoDB

### Build && Setup, may require extra privileges
* Setup (once)
```
python DOCKER.py build
```
* Build
```
python DOCKER.py setup
```
* Run
```
python DOCKER.py run
```
* Kill
```
python DOCKER.py kill
```
